# my-project

node js 설치되어있어야합니다.

## Project setup
디렉토리 생성해서 git clone 받습니다.
받으면 backend, front 폴더가 있습니다.
backend는 관리자 서비스를 위한 서버입니다.
홈페이지를 실행하기 위한 서버는 front 폴더에 있습니다.

front 폴더로 이동후 필요한 모듈을 설치합니다.

```
cd front
npm install
```

### Compiles and hot-reloads for development
로컬 서버를 실행시킵니다.
```
npm run serve
```

실행 후 뜨는 주소로 홈페이지로 접속합니다.
